# Roadmap

### Shipped

**Dynamic Fees** ✓

<mark style="color:$info;">Our core feature. Swap fees that adjust to market conditions, making pools competitive across volatility regimes. Live on Base Mainnet.</mark>

**Unified Yield** ✓

<mark style="color:$info;">Idle liquidity deployed to Aave, earning yield even when not actively used in swaps. Maximizes capital efficiency for LPs.</mark>

**Points System** ✓

<mark style="color:$info;">Rewards for early adopters and active community members. Earn Points through swapping, providing liquidity, and referrals.</mark>

{% hint style="success" %}
All core features are live on Base Mainnet.
{% endhint %}

#### Long-Term Vision

The DeFi landscape evolves rapidly, making long-term predictions challenging. Our north star is **Unified Pools** - single pools with unified liquidity that deliver best-in-class features for all participants.

We'll achieve this vision through continuous innovation, market-driven insights, and strategic partnerships that push the boundaries of what's possible in decentralised finance.
